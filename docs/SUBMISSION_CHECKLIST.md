# Agents League Submission Checklist

Deadline: June 14, 2026, 11:59 PM Pacific Time.

- Register and activate the hackathon profile.
- Select the Reasoning Agents challenge.
- Confirm the app is publicly accessible or straightforward for judges to run.
- Publish a public GitHub repository.
- Include a project description covering the problem, features, and Microsoft technologies.
- Include an architecture diagram showing Microsoft Foundry and IQ usage.
- Record a demo video of no more than five minutes and upload it to YouTube or Vimeo.
- Demonstrate multi-step reasoning, citations, reliability, and clear fallback behavior.
- Do not claim live Foundry IQ or Fabric IQ usage until cloud resources are connected and tested.

## Suggested Demo Story

Ask: "Which revenue risks need executive attention and what policy should we follow?"

Zentro should demonstrate:

1. Planning the compound question.
2. Retrieving live business state through Fabric IQ.
3. Retrieving the escalation policy through Foundry IQ.
4. Combining both into a cited, governed action.

This directly supports the judging criteria for accuracy, multi-step reasoning, user experience, reliability, and the Best use of IQ tools prize.
